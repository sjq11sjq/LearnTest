您还可以按照以下说明从计算机中上传现有文件。


Git 全局设置
git config --global user.name "sjq11sjq"

创建一个新仓库

cd existing_folder
git init
git remote add origin git@gitlab.com:sjq11sjq/sjq-1-project.git
